A little bit about me.....

Surprisingly, my journey into tech started with a degree in English. As I started writing for print publication, I felt compelled to learn the fascinating technology that was part of writing's future. I knew I was a creator. I loved the idea that I could help others express themselves or represent their businesses. It was during an administrative job that required web maintenance

that I fell in love with coding. At that time, I had no idea how to code but I absolutely loved the challenges of running the company website. That event opened a door that could never be shut. I realized I found something that challenged me intellectually and allowed me to create. Now I've made a full switch to full-stack developement, where I can use my editorial and analytical skills to write clean, efficient code.

Hand-Crafted HTML

My focus is writing well formatted, semantic HTML5. I make sure that the code is easy to read, to use in collaboration, to trouble-shoot and to access. I can do the same for your website copy, if you are interested.

Lean and Clean CSS

I pride myself on writing CSS that is easy to read and expand. I focus on keeping my CSS lean and fast to load, and I make it a habit to stay up to date on current best practices.
